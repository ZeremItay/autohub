# הוספת עמודת Status לקורסים

## חשוב: הרץ את קובץ ה-SQL לפני השימוש!

כדי שהפיצ'ר של סטטוס קורסים יעבוד, אתה צריך להריץ את קובץ ה-SQL הבא במסד הנתונים:

**קובץ:** `supabase-add-course-status.sql`

## איך להריץ:

1. פתח את Supabase Dashboard
2. לך ל-SQL Editor
3. העתק את התוכן של `supabase-add-course-status.sql`
4. הרץ את השאילתה

## מה הקובץ עושה:

- מוסיף עמודה `status` לטבלת `courses`
- ערכים אפשריים: `'draft'` (טיוטה) או `'published'` (מפורסם)
- מגדיר את כל הקורסים הקיימים כ-`'published'` כברירת מחדל

## איך זה עובד:

- **קורסים בטיוטה** (`status = 'draft'`): לא יוצגו לקהל הרחב, רק למנהלים
- **קורסים מפורסמים** (`status = 'published'`): יוצגו לכל המשתמשים
- **בפאנל הניהול**: תוכל לבחור את הסטטוס בעת יצירת/עריכת קורס

## הערה:

אם לא תריץ את הקובץ, המערכת תמשיך לעבוד אבל כל הקורסים יוצגו (ללא סינון לפי סטטוס).
